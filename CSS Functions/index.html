    CSS Functions

  • Functions in CSS are reuseable pieces of code which perform specific tasks. Functions are passed 'arguments' between parentheses.
    Ex.
    'color: rgb(0, 42, 255);'
    'background: linear-gradient(90deg, blue, red);'

  • CSS does NOT allow us to create our own functions. The language comes with a list of premade functions.

      Some of the premade CSS functions

    • calc()
    • min()
    • max()
    • clamp()

  • Examples of CSS variables

    • calc()

      --header: 3rem;
      --footer: 40px;
      --main: calc(100vh - calc(var(--header) + var(--footer)));

      --header, --footer, and --main are all examples of CSS variables
      main = 100vh - (header + footer)
      calc() is handing the math even with mixed vh, rem, and px units.

    • min()

      width: min(150px, 100%);
      height: min(150px, 100%);

      If there are 150px available to the image, it will take up all 150px. If there are not 150px available, the image will switch to 100% of the parent’s width.
      You can also do basic math inside min().
      width: min(80ch, 100vw - 2rem);

    • max()

      works the same way as min, only in reverse. It will select the largest possible value from within the parenthese.
      The max function is most useful when the viewing window is either exceptionally small, or the user increases the content size by using the browser’s zoom feature. You may not find a lot of use for max at first, but it is a good tool to be aware of for projects where accessibility is important.

    • clamp()

      clamp() makes elements fluid and responsive.
      font-size: clamp(320px, 80vw, 60rem);

          the smallest value (320px)
          the ideal value (80vw)
          the largest value (60rem)
