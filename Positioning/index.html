Positioning

  1. Static - the default position
  2. Relative - same as static but you can add top, right, bottom, left making it move relative to it's normal position
  3. Absolute - Removes from the flow and positions absolute to the parent. Parent has to be relative, absolute, sticky, or fixed position for top, right, button, left to work else it considers the main html element as the parent
  4. Fixed - fixed to a place. Doesn't give two shits about the parent. Always considers html element the parent. Stays there when scrolled.
  5. Stick - relative - when normal + fixed - when scrolled. The values for top, right, bottom, left become active when scrolled.
